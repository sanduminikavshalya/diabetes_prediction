# Diabetes Prediction using Machine Learning

## Description
This project predicts whether a person has diabetes using medical attributes.

## Dataset
PIMA Indians Diabetes Dataset (Kaggle)

## Technologies Used
- Python
- Pandas, NumPy
- Scikit-learn
- Streamlit

## Model
Logistic Regression

## How to Run
1. Install dependencies
2. Run `python diabetes.py`
3. Run `streamlit run app.py`
